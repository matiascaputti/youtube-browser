# Youtube Browser

### React + Redux

Checkout this repo, install dependencies, then start the gulp process with the following:

```
    > git clone git@github.com:maticaputti/react-redux-youtube-browser.git
    > cd react-redux-youtube-browser
    > npm install
    > npm start
```
